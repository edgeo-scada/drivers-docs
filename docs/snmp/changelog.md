# Changelog

All notable changes to the SNMP client library.

## [1.0.0] - 2024-01-15

### Added
- Initial release
- Full SNMPv1, SNMPv2c, SNMPv3 support
- GET, GETNEXT, GETBULK, SET operations
- MIB walk functionality
- SNMPv3 authentication (MD5, SHA, SHA-224, SHA-256, SHA-384, SHA-512)
- SNMPv3 privacy (DES, AES, AES-192, AES-256)
- Connection pooling with load balancing
- Trap listener
- Built-in metrics
- CLI tool (edgeo-snmp)
