# Changelog

All notable changes to the BACnet client library.

## [1.0.0] - 2024-01-15

### Added
- Initial release
- Full BACnet/IP protocol implementation
- Device discovery with WhoIs
- ReadProperty and WriteProperty support
- ReadPropertyMultiple support
- COV (Change of Value) subscriptions
- BBMD foreign device registration
- Priority array support (1-16)
- Built-in metrics
- CLI tool (edgeo-bacnet)
