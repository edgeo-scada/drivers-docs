# Changelog

All notable changes to the S7 client library.

## [1.0.0] - 2024-01-15

### Added
- Initial release
- Full S7comm protocol implementation
- Support for S7-200, S7-300, S7-400, S7-1200, S7-1500, LOGO!
- Read/write operations for all memory areas (DB, I, Q, M, T, C)
- Multi-item read/write operations
- Connection pooling
- Automatic reconnection with exponential backoff
- Built-in metrics
- CLI tool (edgeo-s7)
